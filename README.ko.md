# 데이트 초대장 메이커

데이트 초대장 메이커는 초대장을 만들고 링크 하나로 공유할 수 있는 서비스입니다.

## 배포 사이트

https://dseung001.github.io/invite-maker/

## 할 수 있는 일

- 초대장 문구 언어 선택: 한국어, 영어, 일본어, 중국어
- 대표 이미지 설정 (샘플 이미지 또는 HTTPS 이미지 URL)
- 가능한 날짜와 시간 선택지 설정
- 시간대 프리셋(점심/오후/저녁)으로 빠르게 시간 채우기
- 선택한 시간대를 다른 날짜로 한 번에 복사
- 음식 선택지 설정
- 초대장 링크 생성 및 공유
- 상대가 최종 선택 결과를 복사/공유해 전달

## 사용 방법

1. 앱 첫 화면을 엽니다.
2. 문구, 이미지, 날짜/시간, 음식 항목을 원하는 대로 설정합니다.
   - 날짜/시간은 프리셋으로 빠르게 선택한 뒤, 필요하면 시간 칩으로 미세 조정합니다.
   - 한 날짜에서 고른 시간대를 다른 날짜로 복사할 수 있습니다.
3. 생성된 초대장 링크를 복사합니다.
4. 상대에게 링크를 보냅니다.
5. 상대는 링크를 열어 단계별로 응답합니다.
6. 마지막 화면에서 결과를 복사하거나 공유해 다시 전달합니다.

## 주의 사항

- 초대장 데이터는 URL 자체에 포함됩니다.
- 링크를 가진 사람은 내용을 확인할 수 있습니다.
- 민감한 개인정보는 입력하지 마세요.
