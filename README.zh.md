# 约会邀请生成器

约会邀请生成器可以让你创建邀请内容，并通过一个链接直接分享。

## 部署地址

https://dseung001.github.io/invite-maker/

## 你可以做什么

- 选择邀请文案语言：韩语、英语、日语、中文
- 设置封面图片（示例图片或 HTTPS 图片 URL）
- 设置可选日期和时间
- 使用时间段预设（午餐/下午/晚上）快速填充时间
- 可将某一天已选时间一键复制到其他日期
- 设置可选食物
- 生成并分享邀请链接
- 让对方复制/分享最终选择结果给你

## 使用方法

1. 打开应用首页。
2. 按需设置文案、图片、日期/时间和食物选项。
   - 日期/时间建议先用预设快速选择，再按需用单个时间选项微调。
   - 你可以把某一天已选择的时间复制到其他日期。
3. 复制生成的邀请链接。
4. 将链接发送给对方。
5. 对方打开链接后按步骤完成选择。
6. 在最终页面复制或分享结果并回传给你。

## 注意事项

- 邀请数据会直接包含在 URL 中。
- 任何拿到链接的人都可以看到内容。
- 请不要填写敏感个人信息。
