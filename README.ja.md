# デート招待メーカー

デート招待メーカーは、招待内容を作成して 1 つのリンクで共有できるサービスです。

## 公開サイト

https://dseung001.github.io/invite-maker/

## できること

- 招待文の言語選択: 韓国語、英語、日本語、中国語
- カバー画像の設定（サンプル画像または HTTPS 画像 URL）
- 候補となる日付と時間の設定
- 時間帯プリセット（昼/午後/夜）で時間をすばやく入力
- ある日付で選んだ時間帯を他の日付へ一括コピー
- 食べ物の候補設定
- 招待リンクの作成と共有
- 相手が最終選択結果をコピー/共有して返信

## 使い方

1. アプリのホーム画面を開きます。
2. 文面、画像、日付/時間、食べ物の候補を設定します。
   - 日付/時間は、まずプリセットでまとめて選び、必要に応じて個別の時間チップで微調整します。
   - 1つの日付で選んだ時間帯を、他の日付へコピーできます。
3. 生成された招待リンクをコピーします。
4. 相手にリンクを送ります。
5. 相手はリンクを開いてステップごとに回答します。
6. 最後の画面で結果をコピーまたは共有して返送します。

## 注意事項

- 招待データは URL 自体に含まれます。
- リンクを知っている人は内容を確認できます。
- 個人情報などの機密情報は入力しないでください。
